Administración y Seguridad en Redes

Mileidy Medina Pajoy
BIENVENIDOS
Objetivo General

Crear un objeto virtual de información a través de tecnologías HTML para adquirir nuevos conocimiento y destreza en el desarrollo de sitios web.

  • Objetivo Especifico 1

    Crear un sitio Web para dar a conocer todo lo referente a la administracion y seguridad de redes.

  • Objetivo Especifico 2

    Identificar el tipo de amenazas de seguridad a las que se expone una red y darlas a conocer mediante informacion textual y audiovisual a traves de un sitio web.

  • Objetivo Especifico 3

    Implementar la temática en el curso Diseño de Sitios Web en la creacion de un sitio donde se evidencie claramente los objetivos alcanzados en el desarrollo del curso.
